## Installation
Please note before importing the pack, the pack is exclusively developed and tested on [Prism Launcher](https://prismlauncher.org/), Any issues that arise from using another launcher will be promptly closed.

1. Go to the [Modrinth Versions Page](https://modrinth.com/modpack/freshstartsmp/versions) and click the latest one (Eg; v26.2.2)
2. Go to the bottom of the page to find "Files"
3. Right Click the "Download" Button and click "Copy Link" (wording may be different depending on your browser, Eg; "Copy Link" on Firefox, "Copy Link Address" on Chrome)
4. Open Prism Launcher and sign in (if you haven't already)
5. In Prism Launcher, Click "Add Instance", Click Import from zip, and paste the link you just copied
6. Launch, Everything else should be sorted.

If any issues arise while using Prism Launcher, please open an [issue](https://github.com/cwbon1/freshstartsmp/issues/new).
