## Installation
Please note before importing the pack, the pack is exclusively developed and tested on [Prism Launcher](https://prismlauncher.org/), Any issues that arise from using another launcher will be promptly closed.

1. Download the pack from [Releases](https://github.com/cwbon1/freshstartsmp/releases/latest)
2. Open Prism Launcher and click "Add Instance"
3. Click "Import from zip" and select the pack you just downloaded
4. Launch, Everything else should be sorted.

If any issues arise while using Prism Launcher, please open an [issue](https://github.com/cwbon1/freshstartsmp/issues/new).
